PRECOMPILED COCKATRICE:

This is a ready-made download of Cockatrice that includes all of the Vault cards. All you need to do is open up the .zip file and run.

LOOSE SET FILES:

If you'd wish to just add the set to an existing Cockatrice download you have, download these. Open up the .zip file, drop the .xml into yourCockatriceFolder\data\customsets, drop the images into yourCockatriceFolder\data\pics\CUSTOM, and then run Cockatrice.
